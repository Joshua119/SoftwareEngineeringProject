Project Description

We are creating an educational game for 4th-6th graders. It is a top down 2D typing game. The game teaches American history and geography. Using Plymouth plantation as a setting. Player character moves around plymouth plantation talking to NPCs and starting minigames. It is a collection of 3 minigames. There are 3 different diffirent difficlty levels each designed for one of the grades. The game is controlled either through WASD or arrow keys. It was made using unity. there will be a scoreboard.
